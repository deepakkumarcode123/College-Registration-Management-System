# College-Registration-Management-System
The College Registration Management System is a desktop-based application developed using Python, Tkinter, MySQL, and TkCalendar. The project enables users to register, update, search, delete, and manage student registration records through an interactive graphical user interface (GUI).  

# 1️⃣ Project_MainPage.py
## **Description**

Project_MainPage.py serves as the core module of the application. It provides a complete Student Registration Form where users can perform various database operations.

## **Key Features**
Student Registration Form
Add New Student Records
Search Student Information
Display All Registered Records
Delete Existing Records
Password Visibility Toggle
Course and City Selection using Dropdown Menus
Gender Selection using Radio Buttons
Date Selection using Date Picker
Integration with MySQL Database
Treeview Data Display
## **Technologies Used**
Python
Tkinter
MySQL (PyMySQL)
TkCalendar
PIL (Python Imaging Library)
Database Operations
INSERT
SELECT
DELETE
# 2️⃣ Update_RegistrationForm.py
## **Description**

Update_RegistrationForm.py is dedicated to updating existing student registration records. Users can modify registration details using their password as a unique identifier.

## **Key Features**
Update Existing Student Records
Modify Personal Information
Update Contact Details
Change Course Selection
Edit City and Gender Information
Password-Based Record Identification
Password Show/Hide Functionality
User-Friendly GUI Interface
## **Technologies Used**
Python
Tkinter
MySQL (PyMySQL)
TkCalendar
PIL (Python Imaging Library)
Database Operations
UPDATE
# 3️⃣ Login_RegistrationForm.py
# **Description**

Login_RegistrationForm.py acts as the authentication module of the system. It verifies user credentials before granting access to the main registration dashboard.

## **Key Features**
Secure Login Interface
Username and Password Authentication
Database Credential Validation
Error Handling for Invalid Login Attempts
Password Visibility Toggle
Direct Navigation to Main Dashboard
## **Technologies Used**
